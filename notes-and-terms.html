Terms

ASCII: American Standard Code for Infomation Exchange

DNS: Domain Name System, AKA the internet phonebook

DOS: Disk Operating System

EBCDIC: (Very old) Extended Binary Coded Decimal Interchange Code (pronounced either "ehb-suh-dik" or "ehb-kuh-dik") is a binary code for alphabetic and numeric characters that IBM developed for its larger operating systems.

High-endian/low(little)-endian: endianness is the order or sequence of bytes of a word of digital data in computer memory. Endianness is primarily expressed as big-endian (BE) or little-endian (LE). A big-endian system stores the most significant byte of a word at the smallest memory address and the least significant byte at the largest. A little-endian system, in contrast, stores the least-significant byte at the smallest address. https://en.wikipedia.org/wiki/Endianness

Hotlinking: pointing your `src` attribute to an image hosted on another site without permission (https://developer.mozilla.org/en-US/docs/Learn/HTML/Multimedia_and_embedding/Images_in_HTML)

IP: Internet protocol

Linkrot: Hyperlinks breaking when the targeted file, web page, or server moves

OEM character set: Original equipment manufacturer character set. A font that uses the extended ASCII characters as defined by IBM for the original PC. the OEM character set is built into every graphics card. It is also the character set used by Windows in order to display DOS applications properly. (https://www.pcmag.com/encyclopedia/term/oem-font)
